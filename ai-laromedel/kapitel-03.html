Kapitel 3

Tekniken bakom AI: Programmering

Grunderna i programmeringslogik med Python – variabler, villkor, loopar, funktioner och algoritmer. Byggstenarna för att förstå och bygga AI-system.

Programspråket Python

Att förstå grunderna i programmering handlar om mer än att bara skriva kod. Det är ett sätt att strukturera tankar och lösa problem systematiskt. Genom att behärska grundläggande logiska byggstenar och utnyttja moderna verktyg kan man gå från att vara passiv användare till att själv skapa intelligenta system.

Många AI-system och maskininlärningsmodeller skrivs i Python. Det är ett programmeringsspråk som är populärt eftersom det är relativt lätt att läsa, men ändå kraftfullt nog för att bygga avancerade program.

Python används i alla faser av AI-arbete:

  • läsa in och rensa data
  • göra beräkningar och statistik
  • bygga algoritmer som hittar mönster
  • träna modeller
  • visualisera resultat i diagram.
Vanliga Python-bibliotek för AI
  • NumPy och Pandas – hanterar tabeller och siffror
  • Matplotlib – ritar grafer och visualiseringar
  • TensorFlow och PyTorch – bygger neurala nätverk
  • Scikit-learn – klassificering, regression och klustring

Ett bibliotek innehåller kod som andra redan skrivit och testat. Du slipper uppfinna hjulet igen.

Grunderna i programmering

Variabler

Variabler är som namngivna lådor där värden sparas. De har ett namn och ett värde, och värdet kan ändras under programkörningen.

namn = "Karin"       # sträng (text)
alder = 17           # heltal (integer)
langd = 1.72         # decimaltal (float)
ar_student = True    # boolesk (sant/falskt)

Det finns flera datatyper: str (text), int (heltal), float (decimaltal) och bool (sant/falskt).

Villkor (if-satser)

Med villkor kan programmet ta olika vägar beroende på om ett påstående är sant eller falskt.

poang = 75

if poang >= 90:
    print("Betyg A")
elif poang >= 70:
    print("Betyg B")
else:
    print("Fortsätt öva!")

Loopar

En loop upprepar ett kodblock ett bestämt antal gånger eller så länge ett villkor är sant. Det är ett grundverktyg för att hantera repetitioner effektivt.

# for-loop: upprepar för varje element i en lista
frukter = ["äpple", "banan", "päron"]
for frukt in frukter:
    print(frukt)

# while-loop: upprepar tills villkoret är falskt
x = 0
while x < 5:
    print(x)
    x += 1

Funktioner

En funktion är ett namngivet, återanvändbart kodblock. Du definierar den en gång och anropar den hur många gånger som helst.

def hälsa(namn):
    return "Hej, " + namn + "!"

print(hälsa("Karin"))   # → "Hej, Karin!"
print(hälsa("Ahmad"))   # → "Hej, Ahmad!"
Skillnad: villkor vs. funktion

En if-sats väljer vilken kod som körs baserat på ett villkor. En funktion är ett namngivet kodblock man definierar en gång och sedan kan anropa upprepade gånger för att slippa skriva samma kod igen.

Algoritmer

En algoritm är en väldefinierad, steg-för-steg-instruktion för att lösa ett problem. Tänk på ett recept: exakta steg, i rätt ordning, som ger ett förutsägbart resultat.

En bra algoritm har tre viktiga egenskaper:

  • Korrekthet – den löser rätt problem och ger rätt svar.
  • Effektivitet – den använder rimliga mängder tid och minne.
  • Tydlighet – den går att förstå, testa och modifiera.

Exempel: algoritm för att hitta det största talet

def hitta_storst(lista):
    storst = lista[0]          # börja med första elementet
    for tal in lista:
        if tal > storst:
            storst = tal       # uppdatera om vi hittar något större
    return storst

print(hitta_storst([3, 7, 2, 9, 4]))   # → 9

Sökning och spelteori

Sökning handlar om att hitta en lösning i ett stort rum av möjligheter. Det är grunden för många AI-tillämpningar, från schackprogram till GPS-navigering.

Breddenförstsökning (BFS) och djupetförstsökning (DFS)

BFS (Breadth-First Search) utforskar alla grannar på en given nivå innan den går djupare. Den hittar alltid den kortaste vägen men kan använda mycket minne.

DFS (Depth-First Search) följer en gren så djupt som möjligt innan den backar. Den är minneseffektiv men garanterar inte kortaste vägen.

A* – informerad sökning

A* (A-stjärna) kombinerar faktisk kostnad med en uppskattning av resterande kostnad (en heuristik). Det gör den effektivare än blind sökning. A* används i GPS-navigering, spel och robotplanering.

Minimax i spel

Minimax är en algoritm för tvåspelarspel. Den antar att motståndaren alltid väljer det bästa draget för sig och väljer det drag som maximerar din minimala vinst. Med alfa-beta-beskärning kan man skippa grenar som aldrig kan bli bättre än redan hittade alternativ – vilket gör sökningen mycket snabbare.

Programmeringsfel och felsökning

Fel i kod kallas buggar. Det finns tre typer:

  • Syntaxfel – koden bryter mot språkets regler och kan inte ens köras.
  • Körtidsfel – koden körs men kraschar under körning (t.ex. division med noll).
  • Logikfel – koden körs utan fel men ger fel svar.
Vanliga fallgropar i Python
  • Listor är nollindexerade: det första elementet har index 0, inte 1.
  • Python skiljer på stora och små bokstäver: Namn och namn är olika variabler.
  • Indragningar (indentering) är obligatoriska – de definierar kodblock.
  • Glöm inte kolonet efter if, for, def, m.m.

Google Colab

Google Colab är ett kostnadsfritt onlineverktyg för AI-programmering. Det kräver ingen installation – du kör Python direkt i webbläsaren. Colab är standard för AI-kurser och forskning, och ger tillgång till kraftfull datorhårdvara (GPU/TPU) i molnet.

Så här kommer du igång:

  1. Gå till colab.research.google.com
  2. Logga in med ett Google-konto
  3. Skapa ett nytt "Notebook" eller öppna ett befintligt
  4. Spara en kopia på din Google Drive via menyn Arkiv → Spara en kopia i Drive
  5. Kör kodceller med ▶-knappen eller Shift+Enter

Sammanfattning

  • Python är det dominerande språket för AI – lättläst och med kraftfulla bibliotek som NumPy, Pandas och TensorFlow.
  • Variabler lagrar data, villkor styr flödet, loopar upprepar kod och funktioner gör koden återanvändbar.
  • En algoritm är en steg-för-steg-instruktion med tre krav: korrekthet, effektivitet och tydlighet.
  • Sökalgoritmer (BFS, DFS, A*, Minimax) är grunden för planering och spelande AI.
  • Google Colab låter dig köra Python i webbläsaren utan installation.
